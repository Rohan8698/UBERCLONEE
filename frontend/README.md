# Uber Clone Frontend

This is the Next.js + Tailwind CSS frontend for the Uber clone project.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```

## Features
- User authentication (Clerk)
- Ride booking UI
- Google Maps integration
- Stripe payment UI
- Responsive design (Tailwind CSS)
